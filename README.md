# 🎮 Tic Tac Toe Game

This is a simple **Tic Tac Toe** game built with [html,css, JavaScript]. It allows two players to play the classic game in a terminal or graphical interface.

## 📌 Features

- Two-player mode
- Clean and simple UI
- Win and draw detection
- Easy to understand code
